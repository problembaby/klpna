자격번호 입력 및 확인 방법

자격번호 입력 방법

  1. 2017년부터 간호조무사 자격이 시·도지사 자격에서 보건복지부장관 자격으로 전환됨에 따라 자격(증)번호 체계 변경

자격증 발급지역이 시·도청인 경우

- 연도가 자격번호에 기재된 경우 : 제2016-12345호

- 연도가 자격번호에 미기재된 경우 : 0000 또는 최초발급연도를 앞에 추가.

제12345호 또는 종로-12345호 → 제0000-12345호 또는 제1989-12345호

자격증 발급지역이 보건복지부인 경우

- 자격증 왼쪽 상단의 일련번호 : 제123456호

  1. 자격증 상 자격번호가 현 자격번호 체계와 다르거나 면허(자격)신고센터 로그인 불가한 경우, 보건복지부 면허민원 안내 홈페이지에서 공인인증서 로그인하여 확인

    ※ 본인 외 확인 불가하며, 정부기관 로그인 시 공인인증서 필수

    ※ 자격번호 확인 후 홈페이지 "마이페이지"상 자격번호 수정요함
