대한간호조무사협회 통합 로그인 기능

다양한 서비스를 한번의 로그인으로 편리하게 이용할 수 있습니다.

여러 서비스에 접속할 때마다 별도의 로그인 할 필요가 없어 사용자의 편의성이 증대되었습니다.

사용자 계정 통합 기능

하나의 로그인 정보 사용으로 편리하게 이용할 수 있습니다.

하나의 로그인 자격증명으로 통합하면 보안이 강화되고, 여러 계정을 추척할 필요성이 줄어듭니다.

최초 로그인 시 실명인증 절차를 거치면서 본인임을 확인해야 합니다.

서비스 통합 로그인, 사용자 계정 통합으로

마이페이지에서 사용자의 정보를 더 쉽고 빠르게 관리할 수 있습니다.

보수교육 또는 직무교육의 교육현황, 대체인정 / 강의실 / 회비, 기금내역 / 나의 게시물 등 사용자의 정보를 마이페이지 한 곳에서 확인하고 관리할 수 있습니다.
